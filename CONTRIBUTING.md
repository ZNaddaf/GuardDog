# Contributing
- Fork, create a feature branch, open a PR.
- Write tests for probes. No shell-outs. No network calls.
- Follow repo code style and commit format: type(scope): message.

## Areas
- /src: launcher + checks engine
- /verifier: verification CLI
- /checks: checks_catalog.json and schema
- /report: self-contained HTML template generator
- /packaging: ISO build and signing scripts
