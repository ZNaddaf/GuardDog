GuardDog – Start Here

  1. Full-format your USB as exFAT.
  2. Copy guarddog.iso, this file, SHA256SUMS, SHA256SUMS.sig to the USB. Create a folder reports.
  3. On the target PC, double-click guarddog.iso to mount.
  4. Open the mounted drive and run GuardDog.exe.
  5. Click Verify. If PASS, click Run Baseline Checks.
  6. When finished, the report opens and is saved under \reports\YYYY-MM-DD_HH-MM-SSZ_HOST\report.html.

Notes: runs read-only, no internet, no admin. If verification fails, do not continue.
