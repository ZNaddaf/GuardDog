GuardDog – Basic Security Check

GuardDog is a small tool that runs a few basic security checks on this Windows computer and writes a report to this USB drive.

What GuardDog does

  • Reads some Windows security settings (like firewall and antivirus status).
  • Does not change any settings.
  • Does not connect to the internet.
  • Saves a report as an HTML file in a reports folder next to this file.

How to use it

  1. Plug this USB drive into the Windows computer you want to check.
  2. Open the USB drive in File Explorer.
  3. Double-click GuardDog.exe.
  4. Wait a short moment; a report file will appear in a reports folder.
  5. Double-click the report file to open it in your web browser.

This is an early version of GuardDog; some checks may not be implemented yet.
